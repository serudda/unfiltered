## What is Browse Fragments?

Browse Fragments is your search interface into your knowledge vault. Find that perfect quote, retrieve a code snippet, or explore fragments by tagâ€”all with a simple command.

## How to Use

Search for fragments using keywords:

```bash frame="none"
/browse-fragments trees
```

### Search Options

| Option       | Description                                 |
| ------------ | ------------------------------------------- |
| `--tag`      | Filter by specific tag                      |
| `--category` | Filter by category                          |
| `--source`   | Filter by source                            |
| `--limit`    | Maximum results to return (default: 10)     |
| `--format`   | Output format: `pretty`, `json`, `markdown` |

### Examples

**Search by keyword:**

```bash frame="none"
/browse-fragments productivity
```

**Filter by tag:**

```bash frame="none"
/browse-fragments --tag wisdom
```

**Combine filters:**

```bash frame="none"
/browse-fragments habits --tag philosophy --category quotes
```

**Export as JSON:**

```bash frame="none"
/browse-fragments --tag javascript --format json
```
